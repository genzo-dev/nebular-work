# Nebular Work

Este projeto foi desenvolvido para a disciplina de Interfaces Ricas, com foco no estudo de Frontend moderno utilizando o framework Angular.

O objetivo é explorar a biblioteca Nebular, entendendo como ela funciona, suas principais funcionalidades e aplicações na construção de interfaces.

## Como rodar:

### 1. Verifique a versão do Node:

O Node precisa ser versão 16 ou menor:
```bash
node -v
```

Se o Node estiver em alguma versão mais alta:
```bash
nvm install 16
nvm use 16
```

### 2. Limpe a instalação e instale as dependências do projeto:

```bash
rm -rf node_modules package-lock.json
npm install
```

### 3. Rode o projeto:

```bash
npx ng serve
```
